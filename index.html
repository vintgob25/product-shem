Продуктовый бриф

Навыки для дохода

15 минут в день на создание доходных активов; первый ощутимый результат за 7 дней; новый артефакт каждую неделю.

Базовые допущения

  • 15-минутная дневная петля привычки с еженедельными отгружаемыми артефактами (WIA).
  • Контент модульный и разложен на skill chips (micro, drill, output).
  • Peer-review с рубриками заменяет большую часть менторского времени.
  • Основной клин — сценарий B (работающий специалист) из-за более быстрого ROI.

Сценарий A

Новичок / смена профессии

  • Момент притяжения: серия отказов/молчания на отклики; «нечего показать».
  • 7-day wow: мини-портфолио + оффер + 10 исходящих + ответ/интро.
  • Риски удержания: стыд, перегруз задачами, страх публичности.
  • Мотиваторы: поддержка группы, маленькие победы, шаблоны и примеры.

Сценарий B (целевой)

Работающий специалист / рост дохода

  • Момент притяжения: перед review/переговорами; стагнация дохода.
  • 7-day wow: перепакованный оффер + пакет услуг/прайс + скрипт переговоров + 10 исходящих.
  • Риски удержания: нет быстрого ROI, нет времени, контент кажется «водой».
  • Мотиваторы: ROI-метрики, быстрые активы, социальный коммит.

Core loops

  1. Ship → Score (rubric) → Social status (Squad board)
  2. Streak → Unlock → Surprise reward
  3. Buddy accountability → completion
  4. Weekly ROI reflection → next sprint selection

Ключевые модули

  • Today (15 min chip)
  • Income Dashboard
  • Weekly Sprint
  • Artifact Builder
  • Squad + Buddy
  • Programs Library

Контентный движок

  • Единица: skill_chip
  • Структура: micro_lesson (3–5 мин) → drill (5–7 мин) → output (2–5 мин)
  • Масштабирование: templates; rubrics; examples; peer_review

Ценообразование

  • Monthly: Basic — 990; Plus — 1990; Pro — 4990.
  • Yearly discount: 25%.
  • Bundle: Income Stack (3 programs) — 3.5 месяцев Plus.

Эксперименты

  1. Mandatory 7-day sprint onboarding → D7 retention, activation
  2. Weekly Income Asset requirement → W2 retention, WAU
  3. Peer-review mandatory vs optional → completion, NPS, cost/user
  4. Buddy vs solo → completion rate, D14 retention
  5. ROI dashboard vs basic progress → streak, WAU
  6. Variable reward on streak day 3 → streak length
  7. Refundable deposit commitment device → D30 retention, churn
  8. Pricing A/B for Plus → ARPPU, conversion
  9. Yearly discount depth test → yearly share, LTV
  10. Program bundle offer → revenue/user, attach rate
  11. Early template gating → time-to-first-asset
  12. Weekly demo-day → W4 retention, referrals
